Comment obtenir les clés API gratuites

🟡 Pixabay — 500 000 images gratuites
  1. Go to pixabay.com
  2. Cliquez S'inscrire — compte gratuit
  3. Go to pixabay.com/api/docs
  4. Copiez la clé sous « Your API key »
  5. Collez dans le champ Pixabay et cliquez Test
🔴 Pexels — 8 000+ images de plantes
  1. Go to pexels.com
  2. Cliquez S'inscrire — compte gratuit
  3. Go to pexels.com/api
  4. Cliquez Get Started — remplissez le formulaire
  5. Copiez la clé, collez-la, puis cliquez Test
🌿 Perenual — base de données plantes
  1. Allez sur perenual.com/docs/api
  2. Créez un compte gratuit
  3. Copiez votre clé API et collez-la ici
✓ Gratuit pour toujours — sans carte bancaire
Vos clés sont sauvegardées automatiquement. iNaturalist et Wikimedia fonctionnent sans clé — totalement gratuit !
